# 桌面文件夹启动器
通过rainmeter实现在电脑桌面上以类似手机文件夹的方式启动程序

如何使用：

首先你需要下载并安装rainmeter：https://www.rainmeter.net/

然后在releases页面下载Desktop Folder Launcher.rmskin

双击安装

安装后在rainmeter里右键Desktop Folder Launcher，打开所在文件夹， 在@resources目录下有shortcuts文件夹，此文件夹用于放快捷方式，同时 可以将这个文件夹的快捷方式添加到桌面，以方便放入你想加入到launcher里 的程序

可以将shortcuts文件夹快捷方式的图标换成透明的，再把Desktop Folder Launcher拖到shortcuts文件夹快捷方式上，这样显得更自然一点

如果想要增加一个Launcher，在右键Desktop Folder Launcher，打开 所在文件夹后，将此文件夹复制，黏贴到上一级再改个你喜欢的名字就好了

打开@resources目录，有一个Usersettings.ini，里面有一些数值可以调

注意快捷方式的最大数量为20，超过12个就会看起来很拥挤，我会在下一个版本 解决这个问题
